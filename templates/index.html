Conclusiones automáticas

  • Esta interfaz mantiene una identidad visual inspirada en el panel que compartiste.
  • El archivo debe incluir exactamente las columnas RUT y CALENDARIO.
  • La aplicación inicia sesión, busca el usuario, ajusta el calendario y genera evidencia de cada fila procesada.
  • Si el sistema expira sesión, el flujo intenta reingresar automáticamente.
